# BuyNow

A minimal Single Page Application for e-commerce transactions built with React, Vite, and TypeScript. It uses `localStorage` for demo auth, catalog, and orders, and includes:
 
- Registration
- Login
- Dashboard (catalog with search)
- Item Detail
- Dummy Payment (creates order)
- Orders list

## Run

```
npm run dev
```

Open `http://localhost:5173` in your browser.

## Notes
- This is a demo; do not use real card info.
- Data persists in the browser via `localStorage`.
- To reset data, clear site storage in your browser.
